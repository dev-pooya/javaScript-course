تمرین 3

  • فرض کنید یک کلاس با ۱۰ دانش آموز وجود دارد
  • برای مدل سازی لیست دانش آموزان کل کلاس از یک آرایه استفاده شده است.
  • هر کدام از دانش آموزان به صورت یک آبجکت (object) مدل سازی شده است.
  • آبجکت یکی از دانش آموزان به عنوان نمونه در زیر آمده است.
    •         student = {
          fullname: "tim wilson",
          age: 25,
          classes: ["html", "js", "css"],
          address: {
            city: "bane",
            street: "emam",
            code: 136,
          },
        }
  • آرایه دانش آموزان مجموعه ای از آبجکت ها میباشد.
    •         students = [ {fullname:'tim wilson', age:25, ...}, {...}, {...} ]

با استفاده از آرایه دانش آموزان که در فایل main.js وجود دارد تکالیف زیر را انجام دهید.

    مرتب سازی (sort)

  1. دانش آموزان را به ترتیب سن مرتب سازی کنید.
  2. دانش آموزان را به ترتیب تعداد کلاس هایی که شرکت کرده اند مرتب کنید. (دانش آموزی که کلاس های بیشتری شرکت کرده در اول نمایش داده شود)
  3. دانش آموزان را به ترتیب حروف الفبای نام خانوادگی مرتب کنید ، اگر نام خانوادگی یکسان بود بر اساس نام کوچک مرتب شوند.

    4. فیلتر کردن (filter)

    👈 برای فیلتر کردن میتوانید از تابعی که در جلسه ۹ آموزش داده شد استفاده کنید

  4. دانش آموزان کمتر از ۲۱ سال را فیلتر کنید
  5. دانش آموزانی که در دوره html شرکت کرده اند را فیلتر کنید
  6. دانش آموزانی که در شهر ماکو زندگی میکنند و حداقل در ۲ کلاس شرکت کرده اند را فیلتر کنید
